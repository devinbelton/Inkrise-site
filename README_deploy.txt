# Inkrise Site Deploy Guide

1. Upload index.html, favicon.svg, og.png to your GitHub repository (inkrise-site).
2. Go to Settings > Pages.
3. Under "Source" select Branch: main and Folder: /(root).
4. Save. Wait a few minutes.
5. Your site will be live at https://YOURNAME.github.io/inkrise-site/
6. Connect your custom domain (inkrise.online) via DNS if desired.
